# Osseus Mongo

[mongoose](http://mongoosejs.com/) based osseus [mongodb](https://www.mongodb.com/) connection module

## Install
```bash
$ npm install osseus-mongo
```

## Usage

#### Configuration

* `OSSEUS_MONGO_URI`
* `OSSEUS_MONGO_USER` - if needed
* `OSSEUS_MONGO_PASS` - if needed
