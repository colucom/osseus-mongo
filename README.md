# Osseus Mongo

Osseus [mongoose](http://mongoosejs.com/) based [mongodb](https://www.mongodb.com/) connection

### Install
```bash
$ npm install osseus-mongo
```

#### Config
config should have a "get" function which receives the key name and returns the value

* OSSEUS_MONGO_URI
* OSSEUS_MONGO_USER
* OSSEUS_MONGO_PASS
